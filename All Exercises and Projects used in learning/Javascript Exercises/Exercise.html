Welcome to your First Exercise in Javascript

For this exercise we will be creating a weight metric converter

We will take in a number in grams(g) and convert it to Kilograms(kg). Right now we only know enough to do this, but later on we will learn how to add logic to this sort of problem!

To complete this exercise do the following:

  • Link this HTML file a .js file
  • Use JS to accept a number input in grams
  • Then in your js file convert this number to Kilograms
  • Afterwards report back in an alert what the weight is in kg
  • Then console.log "Conversion Completed"
